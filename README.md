# Res Transport Voting System (JavaFX)

This project is an application designed to manage a residence transport voting process for students. The system allows students to register, authenticate securely, and vote for their preferred transport option and time slot.

## 🚀 Features
- Student registration with unique student ID
- Secure login authentication
- Transport voting (4 seater, 7 seater, 14 seater, or None)
- Time slot selection for transport
- Vote aggregation and results display
- JavaFX graphical user interface (GUI)
- Object-Oriented Design (OOP principles)

## 🛠 Technologies Used
- Java 17
- JavaFX 17
- Eclipse IDE
- Object-Oriented Programming (OOP)

## 🧠 System Design
The application follows a layered approach:
- **UI Layer**: JavaFX screens (Main, Register, Login, Voting, Results)
- **Logic Layer**: StudentSystem and VotingSystem classes
- **Data Models**: Student and Vote classes

## 📌 Learning Outcomes
- Building GUI applications using JavaFX
- Applying OOP concepts such as encapsulation and separation of concerns
- Event-driven programming
- Transitioning from console-based applications to GUI-based systems

## ▶️ How to Run
1. Install Java 17
2. Download JavaFX 17 SDK
3. Configure VM arguments
     4. Run the `Main` class
